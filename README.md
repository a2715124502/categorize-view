# Cross Stitch 图片分类预览

这是一个十字绣图片素材分类预览页面。

## 文件说明

- `01_标签分类需求/分类预览_新.html` - 分类预览HTML页面
- `01_标签分类需求/我的方案/最终分类完整数据_生产版.csv` - 分类数据CSV文件
- `01_标签分类需求/我的方案/分类预览.html` - 同上（在子文件夹中）

## 本地预览

如果需要在本地预览：

1. 进入 `01_标签分类需求` 目录
2. 使用Python启动简易服务器：
   ```bash
   python3 -m http.server 8080
   ```
3. 打开浏览器访问：`http://localhost:8080/分类预览_新.html`
